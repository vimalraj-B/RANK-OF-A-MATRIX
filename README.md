## Step 4:End the program

## Program:

```
#Program to find the rank of a matrix.
#Developed by: B.VIMALRAJ
#RegisterNumber:212224230304
import numpy as np
A=np.array([[5,-3,-10],[2,2,-3],[-3,-1,5]])
sol=np.linalg.matrix_rank(A)
print(sol)
```
